🎯 Objectif du projet

Ce projet a été développé dans le cadre de mon portfolio étudiant en développement full-stack afin de démontrer mes compétences en :

développement backend avec Django

création d’API REST

développement frontend avec React

conception d’une application web complète

L'application permet aux utilisateurs de :

✅ Suivre leurs dépenses et revenus quotidiens

✅ Organiser leurs transactions par catégories personnalisées

✅ Gérer leurs factures récurrentes ou ponctuelles

✅ Visualiser leurs habitudes financières avec des graphiques

✅ Analyser l’évolution de leurs finances dans le temps

🚀 Technologies utilisées

Backend

Python 3.11

Django 5.2

Django REST Framework

SQLite (facilement remplaçable par PostgreSQL)

Django CORS Headers

Frontend

React 19

Vite

Tailwind CSS

Shadcn/UI

Recharts

Axios

React Router

Lucide React
